Política de Privacidade — EI Construtor

Desenvolvedor: Edson Alves Dionizio (pessoa física)

1. Resumo

O EI Construtor é um aplicativo 100% offline desenvolvido por Edson Alves Dionizio como projeto pessoal de pesquisa e desenvolvimento em tecnologia aplicada à construção civil. Ele NÃO coleta, transmite, vende ou compartilha qualquer informação pessoal ou de uso. Todos os dados que você insere ficam armazenados exclusivamente no seu dispositivo, no armazenamento local do navegador (localStorage).

Não há servidores. Não há banco de dados em nuvem. Não há contas de usuário. Não há anúncios. Não há rastreadores.

2. Quais dados o app processa

Você pode inserir, no app, dados como:

  • Informações de orçamentos e contratos (nome de clientes, endereços, valores)
  • Vistorias técnicas e laudos (medições, observações, fotos)
  • Catálogo de serviços e materiais
  • Dados da sua empresa (CNPJ, CREA, CRT) — caso opte por configurá-los
  • Fotos capturadas durante vistorias

Esses dados são processados localmente no seu dispositivo e NÃO são enviados ao desenvolvedor nem a nenhum servidor.

3. Como os dados são armazenados

Todos os dados ficam em:

  • localStorage do navegador (do seu dispositivo)
  • Eventual backup JSON que você mesmo exporta manualmente

Quando você desinstala o app ou limpa os dados do navegador, todos os dados são apagados permanentemente. Recomendamos backups regulares pelo próprio aplicativo (menu Configurações → Exportar Backup).

4. Permissões do dispositivo

O app pode solicitar as seguintes permissões:

  • Câmera — usada apenas quando você anexa uma foto a uma vistoria, laudo ou orçamento. As imagens ficam armazenadas localmente.
  • Armazenamento local — necessário pra salvar seus dados.

O app NÃO usa: localização, contatos, microfone, identificador de propaganda, biometria, telefone, calendário ou qualquer outro dado sensível.

5. Compartilhamento via Web Share API

Quando você toca em "Compartilhar" um orçamento, contrato ou laudo, o aplicativo abre o menu nativo de compartilhamento do Android (mesmo que outros aplicativos como WhatsApp usam). Esse processo acontece localmente no seu dispositivo — o EI Construtor não tem como saber para quem você compartilhou ou o que aconteceu depois.

6. Crianças

O app é destinado a profissionais adultos da construção civil. Não é direcionado a menores de 18 anos e não coleta intencionalmente dados de crianças.

7. Direitos do titular dos dados (LGPD)

Como os dados ficam exclusivamente no seu dispositivo, o controle é seu. A qualquer momento você pode:

  • Acessar: abrir o app e ver todos os dados
  • Corrigir: editar qualquer registro pelo próprio app
  • Excluir: apagar registros individualmente ou desinstalar o app (apaga tudo)
  • Portabilidade: exportar backup JSON pelo menu Configurações

8. Mudanças nesta política

Eventuais atualizações serão refletidas neste documento, com data atualizada no topo. Como o app é offline, mudanças de política só passam a valer quando você atualiza a versão do aplicativo na Play Store.
